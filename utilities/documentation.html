string boolToFrench(bool ibool) is a function used in the user interface

  • Accepts either true or false as a paremeter
  • Returns a translation of the boolean as a string

Example:

#include <iostream>
#include "define.cpp"
          
int main() {
    Chimie::boolToFrench(true); // returns "Oui"
    Chimie::boolToFrench(false); // returns "Non"
}


void loading(int milliseconds) is a function that serves as a loading screen

This function is very simple and doesn't do much. It is also very easy to call and use. Simply call Chimie::loading(100) where 100 will be the gap between each percentage of the loading in milliseconds.
